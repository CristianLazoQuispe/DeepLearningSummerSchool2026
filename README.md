# Deep Learning for Science Summer School 2026 (DL4SCI)

My notes from the **2026 Deep Learning for Science (DL4SCI) Summer School**, a five-day
intensive program hosted by Computing Sciences at Berkeley Lab.

- **Dates:** July 20–24, 2026
- **Location:** Berkeley, CA
- **Focus this year:** foundation models, reasoning, and agentic AI for scientific discovery

The program spans the end-to-end lifecycle — data, training at scale, adaptation, and
evaluation — alongside sessions on reasoning-centric workflows and agentic systems, with an
emphasis on practical application and best practices for deep learning on HPC systems.

## Topics

- Diffusion Models
- Reasoning
- AI Agents & Systems
- Multi-modal Models
- Deep Learning Performance and Scaling
